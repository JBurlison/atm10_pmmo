# PMMO Item Skill Requirements

This document outlines the skill requirements for different types of items in All The Mods 10.

## Tool Requirements

| Tool Type | Required Skills |
|-----------|----------------|
| Pickaxe | Smithing + Mining |
| Axe | Smithing + Woodcutting |
| Shovel | Smithing + Excavation |
| Hoe | Smithing + Farming |
| Shears | Smithing + Farming (reduced) |
| **Paxel** | Smithing + Mining + Woodcutting + Excavation |

## Armor Requirements

All armor pieces require:
- Smithing (to craft)
- Endurance (to wear)

## Weapon Requirements

| Weapon Type | Required Skills |
|-------------|----------------|
| Swords & Melee | Smithing + Combat |
| Bows & Crossbows | Smithing + Archery |

## Special Item Categories

### Tech Items
Tech items from mods like Mekanism, Thermal, RFTools, Applied Energistics, etc. have an additional **Technology** skill requirement on top of their normal requirements.

Example: A Mekanism Atomic Disassembler would require:
- Smithing (tier level)
- Mining (tier level)
- Technology (tier level)

### Magic Items
Magic items from mods like Botania, Forbidden & Arcanus, Occultism, etc. have an additional **Magic** skill requirement on top of their normal requirements.

Example: An Elementium Sword from Botania would require:
- Smithing (tier level) 
- Combat (tier level)
- Magic (tier level)
